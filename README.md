PodsRepertory
=============

Installation





Edit your podfile

~~~
source 'https://github.com/CocoaPods/Specs.git'
source 'https://github.com/heivin/PodsRepertory.git'
platform :ios,'7.0'
inhibit_all_warnings!
pod 'VLayoutManager'
......
~~~



`pod install --verbose --no-repo-update`

`pod update --verbose --no-repo-update`
